# 🚀 BrandPulse AI

Real-time Social Media Sentiment Analysis Dashboard built using Streamlit.

## 🔥 Features
- Live tweet sentiment analysis
- Interactive dashboard
- Real-time data visualization
- AI-based sentiment classification

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- NLTK
- Scikit-learn
- TensorFlow

## ▶️ How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
```
